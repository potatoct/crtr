WordPress5.8 on GitHub

WordPress5.8 on GitHub

December 28, 2021adminPC, Website, WordPress

Currently this website is hosting on GitHub for free, because GitHub is an opensource platform.

Notice: The website is a static website and not a dynamic website, this can be use for experimental and to test out WordPress without spending money. Not recommended for Business or online store.

Below is the not so easy way for creating your own website without coding=>

Step1: Install WordPress 5.8 on your local machine (offline). Here is the link for the download, and I recommend you to watch the tutorial first before doing the installation because it will help you understand the process, and here is the link of tutorial.

Notice: Please save the Username and Password.

Step2: Install Simply Static on you WordPress site, this plugin allow you to generate and download static HTML file for your website don’t worry no coding involve here, you don’t need to learn HTML. Here is how you install it=>

First go to WordPress

Next type login behind wordpress/

Login with Username and Password

Hover over Plugin and click Add New

Search for Simply Static, install and activate it

Hover over Simply Static click Setting, Under general unchecked everything except these 2 and SAVE CHANGES

Finally go to Simply Static, click Generate Static Files, wait for loading, click Click here to download

Notice: Internet connection is required for the download and don’t forget to extract the file

Step3: Hosting, you need GitHub account and you can create it for free and here is how you do it. After login to your account, create a Repository=>

Click on GitHub logo and click New

Fill the Information, check Public, check Add a README file, click Create repository

Notice: Name and Public is important

Step4A: Install GitHub desktop and pair it with your GitHub account, Here is how it work. Next clone the repository to your local machine, where we can upload your Static HTML file and then push (upload) it to your GitHub account.

Here is how you push (upload)=>

Open GitHub desktop and click on Show in Explorer

Copy all the static file and paste it here and close the folder

Fill information before Commit to main

Click on Push origin and wait until complete and everything is done

Step4B: Install GitHub desktop and pair it with your GitHub account, Here is how it work. Next create local repository, where you can upload your Static file and then push (upload) it to your GitHub account.

Notice: If you go with Step4B then you can skip Step3.

Here is how you create local repository=>

Click on down arrow, hover over Add, and click on Create new repository

Fill in information, check README, and click Create repository

Click on Publish repository and wait until complete

Here how is you push (upload)=>

Open GitHub desktop and click on Show in Explorer

Copy all the static file and paste it and close the folder

Fill information before Commit to main

Click on Push origin and wait until complete and everything is done

Step5/Final step: after push (upload) static file is complete, go the GitHub repository to change some setting required for hosting a web site=>

Go to repository and click on setting

Go down to GitHub Pages and click on Click it out here

Click on None and choose main

Click on Save and wait

When the link turn green it’s ready

Notice: When the link is green click on it to see if it work, you don’t need custom domain (you can buy one if you need it).
